# Vim IDE for Python
## this is my custom configuration to install Vim in Linux

###### Pre Requisites
- cURL
- git


###### steps to install
1. curl https://raw.githubusercontent.com/RodrigoHernan/vim/master/install.sh --output /tmp/install.sh
2. source /tmp/install.sh
