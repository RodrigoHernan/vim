# Vim IDE for Python
## this is my custom configuration to install Vim in Linux

###### Pre Requisites
- cURL
- git


###### steps to install
1. sudo apt install build-essential cmake python3-dev git curl
2. source <(curl -s https://raw.githubusercontent.com/RodrigoHernan/vim/master/install.sh)
